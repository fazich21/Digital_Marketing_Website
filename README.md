# Rankovate (Next.js 16)

This project is built with **Next.js 16** using the **App Router**.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui components

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Local setup

```bash
npm install
npm run dev
```

The app routes are now:

- `/`
- `/about`
- `/contact`
